Build instructions

[Requires Qt-Creator](http://qt-project.org/downloads)

Building LoLUpdater:

For x64 (in the .pro file)
LIBS += -LWINDOWSSDKDIR\\Lib\\winv6.3\\um\\x64\

For x86 (in the .pro file)
LIBS += -LWINDOWSSDKDIR\\Lib\\winv6.3\\um\\x86\
