Download LoLUpdater for League of Legends Patch 4.19

2014-11-13 04:39 GMT+1

Installation:
Close the LoLClient/game
Install the vcredist(s).
Run from any of the paths below (or equivalent, lolupdater downloads a 17Mb file, so be patient.)
Standard LoL directory C:\Riot Games\League of Legends
Garena (32-Bit) C:\Program Files\GarenaLoL\GameData\Apps\LoL
C:\Program Files\GarenaLoLPH\GameData\Apps\LoLPH
Garena (64-Bit) C:\Program Files (x86)\GarenaLoL\GameData\Apps\LoL
C:\Program Files (x86)\GarenaLoLPH\GameData\Apps\LoLPH
Will install on all League/Garena/PBE installations if:
Garena/League installs are not installed in the same directory
or
League/PBE installation does not contain a file called "lol.exe" in the same directory as lolupdater
