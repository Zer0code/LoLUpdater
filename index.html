How to install:
Exit the pvp.net client/game, everything that is league of legends
Install the vcredist(s) (both if you are 64bit, x86 only if you are 32bit).
Run from any of the paths below (your installation directory for LoL)
Standard LoL directory/PBE C:\Riot Games\League of Legends
Garena (32-Bit) C:\Program Files\GarenaLoL\GameData\Apps\LoL
C:\Program Files\GarenaLoLPH\GameData\Apps\LoLPH
Garena (64-Bit) C:\Program Files (x86)\GarenaLoL\GameData\Apps\LoL
C:\Program Files (x86)\GarenaLoLPH\GameData\Apps\LoLPH
Manual Installation:
Download/Install Adobe Air and NvidiaCG
Move cg.dll, cggl.dll and cgd3d9.dll from C:\Program Files (x86)\NVIDIA Corporation\Cg\bin to C:\Riot Games\League of Legends\RADS\solutions\lol_game_client_sln\releases\highest version number\deploy
Copy Adobe AIR.dll and NPSWF32.dll from C:\Program Files (x86)\Common Files\Adobe AIR\Versions\1.0 to C:\Riot Games\League of Legends\RADS\projects\lol_air_client\releases\highest version number\deploy\Adobe AIR\Versions\1.0
Check the highest SMID instruction your computer supports AVX2 (haswell), AVX, SSE2 (most common nowadays), SSE or none, then download the file and put it in the same directory as the cg files, finally rename the file to tbb.dll
To finish everything off make sure you unblock the replaced files by right clicking on the file -> properties -> then click unblock, if you can't see the unblock button then it already is unblocked and you do not have to think about this.
