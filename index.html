Latest feature update: 2014-10-11, 08:40 GMT+1

Development started in May 2014 after I (Loggan) got fed up with low framerates on a average computer at the time.

Command Line Switches:

-install : Installs LoLUpdater with default settings

-uninst : Uninstalls LoLUpdater

-help : Shows the help text

Administrative privileges required

LoLUpdaters requires the .NET Framework 4.0
